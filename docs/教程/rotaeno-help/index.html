「轮椅教程」Rotaeno食用指东

🚨 如若看完本文后你仍旧无法理解其使用方式，请立刻退出本页面并遗忘本页面的内容。

Rotaeno（旋转音律）的残疾人教程 此教程为Android适用，不可用于IOS 皆在指引你下载与购买正版的Rotaeno 对于无法安装谷歌框架的设备，本教程仍旧可用

事前准备

在正式开始本教程之前，你需要核对以下几点。以确保你可以在下载游戏后，正常游玩Rotaeno。

本教程不教您如何使用PayPal付款。

  1. 系统版本
    • Rotaeno是一款2022年正式上线的游戏，它对一些安卓系统具有限制。游戏无法在过低的安卓系统上运行
      • 系统版本要求：Android 8 或更高
  2. 机能要求
    • Rotaeno是需要使用手机的陀螺仪或加速度计才可正常游玩的游戏
    • 请自行查询您的设备是否具有陀螺仪或加速度计。
  3. 内存要求
    • 游戏包体大小 [1.5.1版本] 为930MB，请预留充足的内存空间。
  4. 电子邮箱
    • 本教程需要你注册在不同的网站注册2个或更多的账户，请提前准备好电子邮箱
    • 如若您不知道如何注册电子邮箱，请访问下方网站后自行寻找解决办法 注册电子邮箱 - Search
      • 推荐您注册QQ邮箱，而非网易的163邮箱。
  5. 钱包
    • Rotaeno 并非一款 免费游玩 的游戏
    • 你需要准备 18-20 人民币购买游戏本体
      • 没钱就别浪费时间看教程了，先攒钱吧😉

从零开始下载游戏

众所周知，因为CN广电局木琴趋势的原因，所以 Rotaeno 在国内无法通过正规渠道下载。 必须使用 Google Play 或其他软件下载市场来下载。 因本文需要照顾华为设备这种无法安装谷歌框架的用户，所以使用TapTap国际来下载游戏是最好的选择

学会魔法，下载TapTap国际版​

国际版禁止中国大陆的IP访问，所以便需要VPN的帮助来正常访问。

本文将一步到位全部教会你如何正常开启VPN并打开TapTap国际版。

如果您已有使用VPN的能力，请跳转到下载TapTap国际版并注册账户

注册 VPN 账户​

  1. 点击此处打开网页

  2. 进入首页，点击 「开始使用」以进入注册页面

  3. 自行根据引导注册，这里不再赘述

    • 如若未收到邮箱验证码，请检查邮箱是否拼写错误，并查看邮箱垃圾桶/回收站。
    • 部分邮箱可能会将验证码自动判定为垃圾邮件。

  4. 进入网站首页后，在左侧的列表中点击「购买订阅」，会进入类似于下图的页面

  5. 点击免费套餐下方的「立即订阅」，点击结算订单。即可购买成功

    • 有几点需要说明
      • 免费套餐限速40mbps，实际在下载游戏时的速度大概为400KB-3MB/s。
        • 这对于那些不想付费的人来说已足够使用。
      • 如若嫌弃下载速度过慢，或后续有其他用途可能会耗尽免费套餐的流量。 则可以考虑购买付费套餐。

  6. 点击「 🧭仪表盘」。向下滑动，找到「一键订阅」，点击后复制订阅地址。

  7. 前往我们的QQ频道，下载 Surfboard APK文件并进行安装。

    • 如果可以，请在下一步中不使用系统浏览器进行TAP的下载
      • 使用 Chrome 浏览器可以防止自带浏览器拦截网址。
      • 频道同样提供了 Chrome 的APK包，请自取。

  8. 打开 SurfBoard 点击下方列表的 「配置」

  9. 在「配置」列表后，点击右下角的 + 号，展开导入页面

  10. 点击「从URL导入」

  11. 填入刚才你复制的链接，点击导入。

  12. 如若全部成功，则在「配置」列表会显示如下页面

  13. 点击下方列表的「代理」，进入代理页面

  14. 选择「Auto」

  15. 点击下方列表的「仪表盘」，进入仪表盘页面

  16. 点击右下角的 「运行图标」 以开始运行VPN

    🚨 开始运行后，**系统会弹出是否允许SurfBoard创建VPN的提示，请务必选择确认/允许**。 否则需要你去设置中再找到SurfBoard进行权限的设置。十分麻烦。
    • 成功运行后，仪表盘的上传速度与下载速度会实时发生变化，DNS与活跃连接数也会发生变化。
    • 此时，请将SurfBoard加入手机清理的白名单列表内，避免被后台自动杀死导致加速失效。
      • 如何加入就不再赘述，手机型号众多。请自行搜索以获得最佳解决方法
      • 你可能需要一个好的搜索引擎，推荐使用bing.com。 Where we go to play
      • 百度NMSL
    • 后续需要再次打开VPN时，直接打开 SurfBoard ，在「仪表盘」内点击右下角的运行即可。

下载TapTap国际版并注册账户​

终于！在经理了前面繁琐的步骤后，你离胜利只差一个TAP账户与18块钱了！

下载TapTap国际版

  1. 你可以在频道内获取到Tap国际版的APK包，但通常为老版本。需要更新。
  2. 推荐前往本网址进行APK的下载
    1. 下载时，务必选择 「下载完全版」。
  3. 安装APK，打开。

注册账户与购买游戏

  1. 打开apk后，点击搜索栏左侧的图标进入至登录页
  2. 点击注册，根据引导选择方法注册
    1. 支持使用中国大陆手机号注册，也可以使用邮箱。推荐邮箱，尽量不泄露个人信息。
  3. 登录
  4. 在搜索栏内搜索 Rotaeno
  5. 弹出的搜索列表内点击rotaeno旁边的购买选项，也就是2.49美元
  6. 使用PayPal付款
    1. 因PayPal涉及银行卡，信用卡的绑定，且PayPal给予的引导足够充足，本文不在此讲述PayPal的使用方式
    2. 如若您没有银行卡或其他情况导致PayPal和其他的支付方法都无法进行购买，请自行寻找代购。

游戏下载​

  1. QQ频道将持续提供 Rotaeno 最新版本的APK包，你可以前往频道下载。
  2. 或者你仍旧可以在购买完成后在tap内进行下载
  3. 安装。
  4. 正版验证
    1. 在你打开游戏时，游戏会要求你打开TapTap，选择Tap国际版。
      1. 这里是检测账户是否购买了Rotaeno
    2. 之后等待十几秒。如若Tap没自动从游戏的商店页面给弹回到游戏页面，则在商店页面内点击打开游戏即可。
  5. 开始游玩吧！

教程结束，希望可以帮到你。
